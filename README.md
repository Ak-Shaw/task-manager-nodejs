# Task Manager - Node.js

### Setup

`HTTPS`
```bash
git clone https://github.com/Ak-Shaw/task-manager-nodejs.git
```

`SSH`
```bash
git clone git@github.com:Ak-Shaw/task-manager-nodejs.git
```

`dependencies`
```bash
npm install
```

### Usage

`production`

```bash
npm start
```

`development`

```bash
npm run dev
```

`testing`

```bash
npm run test
```

Refer API Usage [here](rest.http)
